GeoMap

Map-based web app for visualizing location data.

Year2026
ScopeProduct & Frontend Design
ClientGeoMap
Duration6 weeks

The Scenario & Business Problem

The existing tool buried useful location data behind dense tables that were hard to scan. I designed a map-first interface that surfaces the same data visually, making patterns obvious at a glance.

Analysts were spending more time scrolling through rows than actually finding insight — the team needed the map itself to do the work the tables couldn't.

The Design & Strategy Challenge

The goal was to make a data-dense product feel legible at a glance, without stripping out the detail power users rely on. Core objectives:

  • Lead with the Map: Turn tabular location data into an interface built around visual patterns.
  • Design a Scalable System: Ship a component system that holds up as the data set grows.
  • Optimize for Performance: Keep the map fast and responsive even with dense data layers.
